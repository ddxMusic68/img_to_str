# imgToStr
This Python library, imgToStr, is designed to convert images into text. It uses the OpenCV library to convert a given image into a NumPy array, and then converts that array into a string using a set of characters.

Installation
To install this library, simply clone this repository to your local machine and run the following command:

bash
```
pip install -e .
```
Usage
To use the imgToStr function, you can call it with the following parameters:

img: A NumPy array representing the image to be converted.
size: A tuple specifying the size of the output image (default: 
(100, 100)
).
chars: A list of characters to be used for the conversion (default: ['@', '%', '#', '*', '+', '=', '-', ':', '.', ' ']).
reverse: A boolean indicating whether to reverse the order of the characters (default: False).

Here's an example usage:

```
python
import cv2
import numpy as np
from imgToStr import imgToStr
```

# Load an image
img = cv2.imread('test.jpg')

# Convert the image to a string
str_img = imgToStr(img)

# Print the resulting string
print(str_img)
showVid Function
The showVid function provides a way to display a video in real-time using the imgToStr function. It takes the following parameters:

capture: A cv.VideoCapture object representing the video capture device.
chars: A list of characters to be used for the conversion (default: ['@', '%', '#', '*', '+', '=', '-', ':', '.', ' ']).
size: A tuple specifying the size of the output image (default: 
(100, 100)
).
font_size: The size of the font used to render the characters (default: 15).
color: A tuple specifying the color of the characters (default: 
(255, 255, 255)
).
reverse: A boolean indicating whether to reverse the order of the characters (default: None).
Here's an example usage:

```
python
import cv2
import pygame
from imgToStr import showVid

# Open the video capture device
capture = cv2.VideoCapture(0)

# Start the pygame display
showVid(capture)

# Release the video capture device
capture.release()
```
